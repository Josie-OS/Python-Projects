# Python-Projects
Various OSINT and Algebra projects using Python3 (& Ruby/others)

1. Multi-Calculator (Algebra)
Multifunction calculator (basic arithmetic, trig, log, power, factorial, combinations/permutations)
Graphing calculator (plots functions using matplotlib; supports single-variable expressions)
Financial calculator (loan payment, mortgage amortization schedule, future value, present value, ROI)

2. Data Graph Explorer (Algebra)
Load CSV, Excel, or JSON files, inspect columns and basic stats, create interactive plots (scatter, line, histogram, bar, box) using Plotly, filter, group, aggregate, and pivot data, save plots as HTML or static images, runs as a lightweight local web app via Dash (works offline).

3. Internet Wayback Machine Downloader (OSINT)
Download entire websites from the Internet Wayback Machine, using concurrency to speed up the process
